# Literary Portfolio

A comprehensive digital portfolio showcasing literary analysis and reflections on great books.

## Project Structure

```
├── index.html          # Main landing page
├── css/
│   ├── main.css       # Main stylesheet
│   └── pages.css      # Page-specific styles
├── js/
│   └── main.js        # JavaScript functionality
├── images/            # Image assets
└── pages/
    ├── analysis1.html # Literary analysis 1
    ├── analysis2.html # Literary analysis 2
    ├── analysis3.html # Literary analysis 3
    ├── analysis4.html # Literary analysis 4
    ├── analysis5.html # Literary analysis 5
    ├── analysis6.html # Literary analysis 6
    ├── summary.html   # Summary page
    └── reflection.html # Personal reflection
```

## Features

- Multiple literary analyses
- Personal reflections on great books
- Responsive design
- Clean and organized structure

## Viewing the Site

This project is hosted on GitHub Pages. Visit the live site at:
https://j-port.github.io/LiteraryPortfolio/

## Local Development

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/j-port/LiteraryPortfolio.git
   ```

2. Open `index.html` in your web browser
